Technologies

I've highlighted the key technologies used in this solution below (hover over a technology to learn for more info).

In Production

The following technologies are included in the final product.

HTML5

I demonstrate the proper use of clean, well-structured semantic HTML5 to provide excellent reliability, accessibility, and maintainablity.

CSS3

I leverage key features from CSS3 to construct a responsive and engaging design. Media-queries, flexbox, and grid layouts provide a rich, mobile-first experience.

Bootstrap 4

Bootstrap provides a great libary of well-tested components and utilities to get any project off the group without "reinventing the wheel".

jQuery

While jQuery less of a "must-have" than it was in the past, it's still very handy for working with events in the DOM. Since Bootstrap already requires it, I make use of it too.

For Development

The following technologies were used during development but aren't part of the final product.

Webpack

Webpack makes building and bundling super easy. It's my favorite way to tie a powerful suite of modern development tools together into one streamlined build process.

NPM

It's hard to beat NPM for package management and it makes common dev tasks super easy.

Docker

Not completely necessary, but I find containerizing my dev environment allows my to get all the tools I need for a project set up "just so", without breaking other projects.

SASS

Proper variables, mixins, and functions in CSS are just too useful to go without, and SASS is my favorite CSS preprocessor to date.

Autoprefixer

Autoprefixer is super handy for cross-browser compatibiltiy because it automatically adds browser-specific prefixes to CSS so you don't have to.

Node.js

A containerized node.js dev server with hot-reloading makes development a total cinch. It's a must have for me when developing web apps not on a traditional LAMP/LEMP stack.
